# y-lab — Y's Laboratory

> 我是Y，这是我的实验室。

开放式知识工程实验室。不做大而全的产品，做**一个能解决问题的方法、一个能复用的工具、一个能思考的框架**。

## Projects

| 项目 | 状态 | 一句话 | 跑起来 |
|------|------|--------|--------|
| [menu-engine](projects/menu-engine/) | ✅ v0.1 | 市场感知菜单工程 | `python projects/menu-engine/demo/run.py` |
| [cost-predictor](projects/cost-predictor/) | ✅ v0.1 | 食材成本预测（季节性+趋势） | `python projects/cost-predictor/demo/run.py` |
| [complaint-classifier](projects/complaint-classifier/) | ✅ v0.1 | 投诉自动分类引擎 | `python projects/complaint-classifier/demo/run.py` |
| [menu-advisor](projects/menu-advisor/) | ✅ v0.1 | 菜单调整建议引擎 | `python projects/menu-advisor/demo/run.py` |

## CLI

