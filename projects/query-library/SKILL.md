# Query Library (知识图谱查询库)

酒店九站知识图谱的查询用例集合。从图谱中提取特定信息的查询脚本。

## 脚本清单 (15个)

| 脚本 | 查询内容 |
|:-----|:---------|
| `_query_agreement.py` | 协议查询 |
| `_query_bf_slip2.py` | 早餐券查询v2 |
| `_query_breakfast_slip.py` | 早餐券查询 |
| `_query_case7794.py` | 案例#7794详情 |
| `_query_checkin.py` | 入住相关查询 |
| `_query_counter.py` | 前台计数器查询 |
| `_query_fire.py` | 消防事件查询 |
| `_query_fire2.py` | 消防查询v2 |
| `_query_label.py` | 标签查询/统计 |
| `_query_nail.py` | 钉子/异物事件查询 |
| `_query_nail2.py` | 钉子事件v2 |
| `_query_pool.py` | 泳池相关查询 |
| `_query_pool2.py` | 泳池查询v2 |
| `_query_public.py` | 公共事件查询 |
| `_query_storage_stocks.py` | 库存查询 |

## 查询模式

1. **ID查询**: 按案例编号/合同编号查询
2. **主题查询**: 按关键词/标签查询关联事件
3. **时间查询**: 按日期范围查询趋势
4. **关系查询**: 查询实体间的关联关系
