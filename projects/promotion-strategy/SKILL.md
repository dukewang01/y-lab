# Promotion Strategy (促销策略引擎)

酒店促销活动分析与策略引擎。基于历史和当前数据，评估促销效果、制定最优促销方案。

## 脚本清单 (8个)

| 脚本 | 功能 |
|:-----|:-----|
| `_618_analysis.py` | 618大促效果分析 |
| `_may_june_strategy.py` | 5-6月整合促销策略 |
| `_may_promo.py` | 5月促销方案分析 |
| `_full_dashboard.py` | 营收全景看板 v1 |
| `_full_dashboard_v2.py` | 营收全景看板 v2 (增强版) |
| `_full_panorama.py` | 九站全景图生成 |
| `_master_report.py` | 总控报告生成 |
| `_compare_0523_0524.py` | 两日数据对比分析 |

## 促销分析框架

| 维度 | 分析内容 |
|:-----|:---------|
| 历史促效 | 2017-2021五年促销数据 (FB图谱) |

| 节日/事件 | 618/情人节/国庆/圣诞 |
