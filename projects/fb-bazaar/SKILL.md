# F&B Bazaar (市集与零售运营)

> B1传媒广场市集 — 酒店最特别的一个"店"

市集（Bazaar）是酒店F&B中的零售业态，卖的是商品而不是餐食。从节日礼盒到日常零售，是酒店收入多元化的试验田。

## 脚本 (5个)

| 脚本 | 功能 |
|:-----|:-----|
| `_check_bazaar.py` | 市集营收数据检查 |
| `_import_takeout.py` | 外卖数据整合（市集部分） |
| `_bubble_analysis.py` | 气泡水/饮品等品类分析 (9.1KB) |
| `_bubble_analysis_v2.py` | 品类分析增强版 (7.8KB) |
| `_fix_bubble.py` | 品类数据修复 |

## 市集品类

| 品类 | 说明 |
|:-----|:------|
| 节日礼盒 | 中秋月饼/春节年货/端午粽子 |
| 烘焙面包 | 每日新鲜出品 |
| 饮品 | 气泡水/果汁/咖啡 |
| 伴手礼 | 苏州特色/酒店定制 |
| 季节性商品 | 圣诞姜饼/新年糖果 |

## 分析维度

- **节日vs日常**: 平日零售 vs 节日爆发
- **毛利率**: 标品 vs 自制产品
- **库存周转**: 保质期管理
- **陈列优化**: 热区 vs 冷区商品
