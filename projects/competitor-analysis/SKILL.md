# Competitor Analysis (竞品分析)

酒店竞品分析系统。基于GCM(集团管理费)数据和公开信息，对比金鸡湖五大酒店竞品表现。

## 脚本清单 (6个)

| 脚本 | 功能 |
|:-----|:-----|
| `_analyze_gcm_peers.py` | 竞品对比分析 (金鸡湖五店) |
| `_analyze_gcm_scenarios.py` | 场景模拟分析 |
| `_analyze_gcm_ytd.py` | GCM YTD趋势分析 |
| `_analyze_gcm_deep3.py` | GCM深度分析v3 (12KB，最全面) |
| `_import_gcm_ytd.py` | GCM YTD数据导入FIN图谱 |
| `_inspect_gcm.py` | GCM数据审视 |

## 竞品范围 (金鸡湖六店)

1. 苏州希尔顿 (自店)
2. 苏州洲际
3. 苏州凯悦
4. 苏州W
5. 苏州中茵皇冠
6. 苏州香格里拉

## 分析维度

- Occ/ADR/RevPAR对标
- 客房收入 vs F&B收入结构
- 季节性对比
- 重大活动/节假日影响

## GCM数据

GCM（Group Contribution Margin — 集团管理费）数据来自业主会议报告，含竞品酒店的YTD经营数据。
