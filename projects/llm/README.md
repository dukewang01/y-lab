# LLM (大语言模型应用)

> **AI在酒店场景的落地实验** — 从闲聊到生产力。

## 探索方向

| 方向 | 说明 | 优先级 |
|:-----|:------|:-----:|
| 🗣️ 智能客服 | 基于九站知识的酒店问答机器人 | ⭐⭐⭐ |
| 📊 报表解读 | DRR/HF报告自然语言解读 | ⭐⭐⭐ |
| 🍽️ 菜品推荐 | CRM偏好→FB图谱→个性化推荐 | ⭐⭐ |
| 📝 自动简报 | 每日营收/投诉/QA自动摘要 | ⭐⭐ |
| 🔍 智能搜索 | 知识图谱的自然语言查询 | ⭐⭐ |
| 💡 策略建议 | 基于历史数据的运营建议 | ⭐ |

## 技术架构

```
用户提问
    │
    ▼
意图识别 (classifier)
    ├── 营收查询 → 查询FIN图谱
    ├── 投诉分析 → 查询GSM图谱
    ├── 菜品推荐 → 查询FB+CRM图谱
    └── 运营建议 → 跨站推理引擎
    │
    ▼
上下文组装 (context builder)
    │
    ▼
LLM回答 (response generator)
```

## 基础设施

- LangChain / 自定义Agent框架
- 九站知识图谱作为RAG数据源
- 飞书/微信作为界面
