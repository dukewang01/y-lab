# y-fsaa — 酒店食品安全审计自动化系统

> **一句话定位：** 酒店食品安全审计自动化和问题追踪，让帮帮报告从手工填表变成一键生成。

---

## 痛点

| 现状 | 问题 |
|------|------|
| 📝 帮帮报告靠手工填 | 每次审计大量手写，效率低、易出错、难追溯 |
| 🔍 整改追踪靠人盯 | 发现问题后靠邮件/微信跟踪，没有系统化管理 |
| ⏰ 超期无人预警 | 整改到期无人提醒，问题项石沉大海 |
| 📊 趋势无数据 | 历史问题不沉淀，无法做趋势分析和预防 |

---

## 系统架构

```
y-fsaa
├── checklist/         # 自检清单管理
│   ├── 创建/编辑检查清单模板
│   ├── 按品类/部门/区域分类
│   └── 支持权重评分
│
├── violation/         # 问题录入 + 整改追踪
│   ├── 违规项录入（照片/描述/责任人/等级）
│   ├── 整改任务分配 + 进度追踪
│   └── 超期自动预警
│
├── report/            # 帮帮报告生成
│   ├── 一键生成审计报告（PDF/Excel）
│   ├── 打分汇总 + 问题明细
│   └── 整改建议自动匹配
│
└── trend/             # 趋势分析
    ├── 历史问题统计
    ├── 部门/品类维度分析
    └── 风险预警看板
```

## 用法示例（Python）

```python
from fsaa_core import AuditEngine, CheckItem

# 创建审计引擎
engine = AuditEngine("中厨房月度审计")

# 添加检查项
engine.add_item(CheckItem("冰箱温度", area="冷库", result=False, severity="严重",
                         责任人="张三", 建议="调整温控设备，增加每日2次温度记录"))

engine.add_item(CheckItem("员工健康证", area="热菜间", result=False, severity="中等",
                         责任人="李四", 建议="健康证到期前30天启动续期流程"))

# 生成报告
report = engine.run()
print(report.to_text())
```
