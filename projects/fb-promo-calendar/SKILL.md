# F&B Promo Calendar (餐饮促销日历)

> 五年促销数据告诉你：什么促销真的有用。

FB图谱里沉淀了2017-2021年的全部促销数据——781个促销产品、101个效果结果、总营收¥19.3万。这个技能把这些历史经验变成决策工具。

## 脚本 (3个)

| 脚本 | 功能 |
|:-----|:-----|
| `_fix_cases.py` | 促销案例修复/整合 (11.8KB) |
| `_import_fb_module.py` | FB模块导入（含促销数据） |
| `_generate_merged_storage_report.py` | 合并存储报告生成 (14.9KB) |

## 促销分析维度

**效果评估:**
```
促销ROI = (促销期增量营收 - 促销成本) ÷ 促销成本
折扣弹性 = 销量增长率 ÷ 折扣率
```

**历史规律（已知）:**
- 客房底线价¥688
- 618/双11/节假日效果最佳
- 客房+餐饮打包 > 单独促销
- 外卖满减 > 直接折扣

**促销日历结构:**
```
1月 元旦/春节礼盒
2月 情人节套餐
5月 劳动节/母亲节
6月 618大促/毕业季
9月 中秋/国庆预热
12月 圣诞/跨年
```
