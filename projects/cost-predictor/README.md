# cost-predictor — 食材成本预测

**Status: Planning** | **Target: v0.1**

基于历史采购数据和外部市场指数，预测下月食材价格变动。

## 要解决的问题

> "牛肉下个月会涨多少？应该提前囤货吗？"

酒店采购成本波动直接影响菜单毛利率。如果能提前1-2个月知道价格走势，就能在涨价前进货、或者提前调菜单。

## 数据来源

```
内部：FIN站历史采购价（月度/单品）
外部：全国农产品批发市场价格指数
外部：季节性因子（休渔期/养殖周期/气候）
```

## 方法路线

| 阶段 | 方法 | 输出 |
|------|------|------|
| v0.1 | 时间序列：ARIMA / 移动平均 | 单品下月价格区间预测 |
| v0.2 | + 外部因子（季节性指数） | 带置信度的预测 |
| v0.3 | + ML（LightGBM） | 多因子预测模型 |

## 用法

```python
from cost_predictor import predict

result = predict("牛肉", months=1)
print(result)
# {"name": "牛肉", "current": 92, "predicted": 98, "range": [95, 102]}
```
