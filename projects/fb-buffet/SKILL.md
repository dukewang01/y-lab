# F&B Buffet (自助餐运营·OPEN专项)

> OPEN是F&B的营收之王。管好OPEN，管好半个餐饮部。

OPEN自助餐是希尔顿F&B营收最大店（月均¥64-85万），也是最大的成本和浪费来源。需要一个专项技能。

## 脚本 (3个)

| 脚本 | 功能 |
|:-----|:-----|
| `_calc_open_takeout.py` | OPEN运营分析（堂食+外卖） |
| `_import_fb_detail.py` | 详细数据导入 |
| `_calc_fb_breakeven.py` | 盈亏平衡分析 |

## 运营看板

**日指标:**
- 餐位数 / 入座数 / 翻台率
- 人均消费（早/午/晚）
- 食材成本率
- 浪费量估算

**周指标:**
- 工作日 vs 周末差异
- 特殊活动日（节日/促销）
- 备餐准确率

**月指标:**
- 营收趋势
- 各时段占比变化
- 顾客组成分析（住客 vs 外部）

## 预测模型

```
明日用餐人数 = 历史同日平均 + 当日预订量 + 特殊事件调整
备餐量 = 预测人数 × 人均取餐量 × 1.1(安全系数)
```
