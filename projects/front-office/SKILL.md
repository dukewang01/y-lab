# Front Office Analysis (前厅运营分析)

酒店前厅运营数据分析工具箱。包含入住率结构分析、节奏分析、数据校验。

## 脚本清单 (4个)

| 脚本 | 功能 |
|:-----|:-----|
| `_analyze_occ_structure.py` | 入住率结构分析（散客/团队/Condo占比） |
| `_analyze_pace.py` | 预订节奏/提前天数分析 |
| `_quick_occ_check.py` | 入住率快速校验 |
| `_check_may_data.py` | 5月数据完整性校验 |

## 分析维度

1. **入住率结构**: 散客 vs 团队 vs Condo vs 免费房
2. **ADR分化**: 各渠道ADR差异（OTA/直客/协议/团队）
3. **预订节奏**: 提前预订天数分布、取消率
4. **Occ趋势**: 周度/月度/季节性规律
