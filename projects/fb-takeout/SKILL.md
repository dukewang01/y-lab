# F&B Takeout (外卖运营分析)

> 不做堂食的补充，做独立的外卖生意

酒店外卖从"歇业时的无奈选择"变成了独立营收来源。这个技能帮你把外卖管明白。

## 脚本 (3个)

| 脚本 | 功能 |
|:-----|:-----|
| `_calc_open_takeout.py` | OPEN自助外卖分析 — 订单/收入/配送费/转化 |
| `_import_takeout.py` | 外卖数据导入图谱 |
| `_calc_fb_breakeven.py` | 盈亏平衡计算（含外卖场景） |

## 分析维度

- **渠道对比**: 美团 vs 饿了么 vs 酒店直营
- **菜品转化**: 浏览→下单→复购的各环节
- **配送费优化**: 满减策略 vs 配送门槛
- **时段分析**: 午市/晚市/周末外卖量
- **外卖 vs 堂食**: 重叠菜品定价差异

## 数据来源

FB图谱中外卖模块 + DRR日报各店数据
