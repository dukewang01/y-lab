# quiz (知识测验)

> **酒店运营知识测试** — 不只是考你，是在帮你建立心智模型。

## 测验范围

| 模块 | 题量(估算) | 内容 |
|:-----|:---------:|:-----|
| 财务营收 | 30 | DRR解读 / 预算对比 / GOP |
| 餐饮运营 | 40 | 各店成本率 / 促销ROI / 菜单工程 |
| 客户管理 | 20 | CRM标签 / RFM / 客群分层 |
| 投诉处理 | 30 | 分类 / 赔偿阶梯 / 法律框架 |
| 食品安全 | 20 | FSAA要点 / HACCP / 虫害 |
| 品牌标准 | 20 | QA条款 / 整改流程 |
| 工程机电 | 20 | HVAC / 电梯 / 消防 |
| 综合运营 | 20 | 跨站联动 / SOP / 收益管理 |

## 题库来源

所有题目从九站知识图谱和 y-lab 实战经验中自动生成。
