# PEST Analysis (外部环境分析)

酒店外部环境分析工具箱。包含PEST分析报告生成、光学/产业链分析、自助餐成本分析。

## 脚本清单 (7个)

| 脚本 | 功能 |
|:-----|:-----|
| `_pest_full_analysis.py` | PEST全量分析报告生成 |
| `_pest_full_analysis_v2.py` | PEST分析v2（增强版） |
| `_pest_real_cases.py` | 真实案例分析库 |
| `_generate_optical_word.py` | 生成光学行业报告 |
| `_optical_cycle.py` | 光学景气周期分析 |
| `_optical_full_chain.py` | 光学全产业链分析 |
| `_qa_pest_report.py` | PEST报告质量校验 |

## PEST框架

| 维度 | 分析内容 |
|:-----|:---------|
| **P**olitical | 政策法规、税收、贸易政策 |
| **E**conomic | GDP、CPI、消费趋势、旅游业 |
| **S**ocial | 人口结构、消费习惯、健康意识 |
| **T**echnological | 数字化、AI、IoT在酒店的应用 |

## 输出

分析报告存储于 `knowledge_center/pest/`
