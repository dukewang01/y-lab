# y-templates — 酒店运营模板库

投诉回复模板、菜单调整方案模板、审计报告模板等。

## 模板列表

| 模板 | 用途 |
|------|------|
| 投诉回复模板 | GSM站×9个场景的标准化回复 |
| 菜单调整方案 | 给总经理的菜单建议报告格式 |
| FSAA审计报告 | 帮帮报告格式 |
| QA自检清单 | 75个标准自检表 |
| 风险登记册 | 风险矩阵填写模板 |
| 员工排班表 | 基于需求的排班模板 |

## 用法

```bash
cat templates/complaint_reply.md
# 复制内容，填入实际情况即可使用
```
