# F&B Cost Control (餐饮成本监控)

> 赚一万不如省三千。成本是利润的第一道门。

酒店餐饮三座成本大山：食材、人工、能源。这个技能专注食材成本监控。

## 脚本 (3个)

| 脚本 | 功能 |
|:-----|:-----|
| `_calc_fb_breakeven.py` | 各店盈亏平衡分析 |
| `_import_beverage.py` | 酒水成本/销售数据导入 (6.3KB) |
| `_import_fb_detail.py` | 餐饮详细成本数据导入 |

## 成本分析框架

```
食材成本率 = 食材成本 ÷ 营收
利润贡献 = 营收 - 食材成本 - 人工分摊
盈亏平衡点 = 固定成本 ÷ (1 - 变动成本率)
```

## 各店成本看点

| 店 | 成本重点 | 监控指标 |
|:-:|:---------|:---------|
| OPEN自助 | 海鲜/牛肉原料价格波动 | 人均食材成本 |
| YUXI中餐 | 高端食材(燕鲍翅)占比 | 毛利率 |
| BACIO意餐 | 进口奶酪/橄榄油 | 成本率趋势 |
| 宴会 | 套餐成本结构 | 单席成本 |

## 数据源

FIN站成本模块 + 采购订单数据 + 供应商报价
