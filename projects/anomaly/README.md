# anomaly (异常检测)

> **检测酒店运营中的异常信号** — 在它变成问题之前发现它。

酒店的每一个数字都在说话。入住率突然跌了10%，食品成本率异常飙升，某个投诉类型在一周内爆发了3次——这些都是异常信号。

## 检测范围

| 维度 | 检测内容 | 数据源 |
|:-----|:---------|:-------|
| 📊 营收异常 | 单日/周度 RevPAR 突变 | DRR日报 |
| 🍽️ 餐饮异常 | 各店营收/成本率偏离 | FB图谱 |
| 📉 Occ异常 | 入住率/ADR 季节性偏离 | 前厅数据 |
| 💰 成本异常 | 食材成本率超阈值 | FIN成本模块 |
| 📢 投诉异常 | 某类投诉突发增多 | GSM图谱 |
| 🛠️ 工程异常 | 设备故障频率 | MEP图谱 |

## 检测方法

- **Z-Score**: 超过均值±3σ标记为异常
- **移动平均**: 7日移动均线偏离度
- **同比对比**: 今年 vs 去年同期偏差
- **规则引擎**: 硬阈值检测（如成本率>40%）
