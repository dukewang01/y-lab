# menu-advisor — 菜单调整建议引擎

**Status: Planning** | **Target: v0.1**

根据成本波动 + 客户偏好 + 投诉情绪，自动给出下月菜单调整方案。

## 要解决的问题

> "下个月菜单应该怎么调？"

菜单工程（menu-engine）回答了"现在怎么样"，menu-advisor回答"下个月怎么办"。

## 输入 → 输出

```
[成本趋势] → 哪些食材涨了 → 建议提价或换食材
[偏好风向] → 哪些口味在流行 → 建议新增方向
[投诉情绪] → 哪些菜常被投诉 → 建议改良或下架
[菜单健康] → 哪些是明星/瘦狗 → 建议推或砍
```
```
              ↓
        [菜单调整建议]
        1. 香煎牛肋骨：提价8元（牛肉+14%）
        2. 新增酸辣风格前菜（偏好+23%）
        3. 改良清蒸老虎斑（投诉率上升）
        4. 下架油炸前菜系列（瘦狗5个月）
```

## 方法路线

| 阶段 | 方法 | 输出 |
|------|------|------|
| v0.1 | 规则引擎（if-else+阈值） | 自动建议清单 |
| v0.2 | + 多因子的加权评分 | 带优先级的建议 |
| v0.3 | + A/B测试建议生成 | 多种方案对比 |

## 用法

```python
from menu_advisor import advise

report = advise(outlet="OPEN", month="2026-07")
for suggestion in report:
    print(f"{suggestion['priority']}: {suggestion['action']}")
```
